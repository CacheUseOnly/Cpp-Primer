- Q:  What values do these literals represent? What type does each have? 
    (a) "Who goes with F\145rgus?\012" 
    (b) 3.14e1L 
    (c) 1024f 
    (d) 3.14L

- A:
    (a) string literal
    (b) long float (long double)
    (c) float (ERROR: The suffix f is valid only with floating point literals)
    (d) long (long double)
